# AI Voice Assistant Chatbot

## Overview
This project is a portable AI voice chatbot that can be embedded into any website using a simple script. It supports natural conversation, emotion detection, multilingual interaction, and simulated business integrations.

## Features
- Voice input and voice response
- Natural conversation AI
- Emotion detection
- Language detection
- Fraud detection (simulated)
- CRM integration (mock data)
- ERP inventory check (mock data)
- Portable chatbot widget
- Can be embedded in any website

## Tech Stack
- HTML
- CSS
- JavaScript
- AI API (for responses)
- JSON (mock database)

## Goal
To build a scalable AI assistant that can be integrated into websites for real-time intelligent customer interaction.

## Future Scope
- Real CRM integration
- Real WhatsApp API integration
- Database connection
- Authentication
- Cloud deployment
